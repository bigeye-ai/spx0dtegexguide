# 🎭 假动作识别 | Fakeout Detection

> **适用场景：** 所有市场状态 | 避免被虚假信号收割

---

## 🎯 核心理念

市场上有三类玩家：

1. **做市商**：他们要对冲，动作有迹可循
2. **机构交易员**：他们有策略，会制造假象
3. **散户（我们）**：容易被假动作骗到

**假动作 (Fakeout)** 是指价格短暂突破关键水平，引诱交易者入场，然后迅速反转，收割止损。

```
价格突破阻力 ──→ 散户追多 ──→ 价格反转 ──→ 散户止损
      ↓
   这就是假动作
```

---

## 🔍 假动作的常见类型

### Type 1: 突破陷阱 (Breakout Trap)

**场景：** 正 Gamma 环境下的假突破

```
        阻力位
           ↓
    ────────●────────
            ↑\
           /  \
          /    \  ← 突破后立即回落
         /      \
    ────●────────●────
```

**识别特征：**
- GEX 仍为正值（橡皮筋效应还在）
- 突破缺乏成交量配合
- 突破后 5-15 分钟内回落

### Type 2: 假翻转 (Fake Flip)

**场景：** GEX 短暂穿越零点

```
GEX 值：
    +5B → +1B → -0.2B → +2B → +4B
                  ↑
            假翻转！
            
    很多人在这里切换策略，被反向收割
```

**识别特征：**
- GEX 仅短暂进入负区间
- 价格没有持续远离翻转点
- 30 分钟内 GEX 翻回

### Type 3: 扫止损 (Stop Hunt)

**场景：** 价格刺穿关键水平，触发止损后反转

```
Put Wall: 5750

价格走势：
    5780 → 5760 → 5745 → 5770 → 5800
                    ↑
              刺穿 Put Wall
              触发多头止损
              然后反转上涨
```

**识别特征：**
- 刺穿幅度小（5-15 点）
- 刺穿时间短（几分钟）
- 之后快速反向

### Type 4: 新闻假动作 (News Fakeout)

**场景：** 重大新闻发布后的第一波反应

```
CPI 数据发布

第一反应：暴跌 50 点 → 散户追空
真实走势：反转上涨 100 点 → 空头爆仓
```

**识别特征：**
- 发生在重大事件后 5-15 分钟
- 波动剧烈但缺乏方向持续性
- "先假后真"是常态

## 🛡️ 防御策略

### 策略 1: 等待确认

**规则：** 不在突破/跌破的瞬间入场

```
错误做法：
    价格突破 → 立即入场

正确做法：
    价格突破 → 等待回踩确认 → 再入场
    
等待时间：
    - 日内交易：5-15 分钟
    - 波段交易：收盘确认
```

### 策略 2: 多重确认

**规则：** 至少 2 个条件同时满足

```
入场检查清单：
□ 价格突破关键水平
□ GEX 数据支持（正 Gamma 回归/负 Gamma 趋势）
□ 成交量放大
□ 相关品种联动（如 QQQ/IWM 同向）

勾选 ≥ 3 个 → 可以入场
勾选 < 3 个 → 继续等待
```

### 策略 3: 聪明止损

**规则：** 止损放在"扫不到"的位置

```
常见止损位置（容易被扫）：
- 整数关口：5800, 5750
- Put Wall / Call Wall 附近
- 前高/前低

更安全的止损位置：
- 关键位外 10-15 点
- 结合 ATR（平均真实波动幅度）
- 非整数点位
```

### 策略 4: 时间过滤

**规则：** 某些时段假动作概率高，避免操作

```
高假动作时段（谨慎）：
- 开盘前 15 分钟
- 重大数据发布后 15 分钟
- 午餐时段 (12:00-13:00 EST)
- 收盘前 15 分钟

低假动作时段（相对安全）：
- 10:00-11:30 EST
- 14:00-15:30 EST
```

## 📊 GexBot 辅助识别

### 利用 GEX 数据过滤假信号

| 价格行为 | GEX 信号 | 判断 |
|---------|---------|------|
| 突破 Call Wall | GEX 仍强正 | ⚠️ 可能假突破 |
| 突破 Call Wall | GEX 转负 | ✅ 可能真突破 |
| 跌破 Put Wall | GEX 仍强正 | ⚠️ 可能假跌破 |
| 跌破 Put Wall | GEX 转负 | ✅ 可能真跌破 |

### Wall 的"黏性"程度

```
强 Wall（不易被突破）：
- OI（持仓量）集中度高
- 距离当前价格较近
- GEX 值较高

弱 Wall（容易被假突破）：
- OI 分散
- 距离当前价格较远
- GEX 值接近 0
```

## 🧠 心理建设

### 被假动作骗了怎么办？

```
错误反应：
被骗 → 愤怒 → 报复性交易 → 亏更多

正确反应：
被骗 → 接受 → 记录复盘 → 调整策略 → 继续
```

### 接受假动作是游戏的一部分

| 心态 | 描述 |
|-----|------|
| ❌ 错误心态 | "我一定能识别所有假动作" |
| ✅ 正确心态 | "假动作会发生，我只需控制损失" |

**关键认知：**
- 假动作是做市商对冲行为的副产品
- 没有 100% 识别假动作的方法
- 好的风控比完美的预测更重要

```markdown
- [ ] 新闻假动作

**具体情况：**

- 下次如何避免？
```

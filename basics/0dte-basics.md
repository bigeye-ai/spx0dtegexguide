# 📚 末日期权入门 | 0DTE Basics

> **新手必读**：什么是 0DTE，为什么要做 0DTE

---

## 🎯 什么是 0DTE

**0DTE = Zero Days to Expiration = 当天到期的期权**

```
今天是周一，这些期权都是 0DTE：
- 周一到期的 SPX 期权 ← 0DTE
- 周一到期的 SPY 期权 ← 0DTE

这些不是：
- 周三到期的 SPX 期权 ← 2DTE
- 下周五到期的 SPX 期权 ← 多 DTE
```

---

## 📅 SPX 期权到期日

SPX 现在几乎每天都有期权到期：

| 日期 | 到期类型 |
|-----|---------|
| 周一 | Weekly |
| 周二 | Weekly |
| 周三 | Weekly |
| 周四 | Weekly |
| 周五 | Weekly + Monthly（每月第三个周五） |

> 💡 这意味着每个交易日你都可以交易 0DTE！

---

## 💰 为什么做 0DTE

### 优点

| 优点 | 解释 |
|-----|------|
| **成本低** | 时间价值快速衰减，期权便宜 |
| **杠杆高** | 小资金可以获得大波动 |
| **风险有限** | 买方最多亏完权利金 |
| **日内结算** | 不用担心隔夜风险 |
| **流动性好** | SPX 0DTE 交易量巨大 |

### 缺点（风险）

| 风险 | 解释 |
|-----|------|
| **时间杀手** | Theta 衰减极快，做错方向很快归零 |
| **波动剧烈** | Gamma 极高，价格变化快 |
| **容易上瘾** | 像赌博一样刺激 |
| **需要盯盘** | 不适合无法看盘的人 |

## 📊 0DTE 的独特特性

### Gamma 集中效应

```
常规期权的 Gamma：
|     /\
|    /  \
|   /    \
|__/______\__ → 行权价
     ATM

0DTE 期权的 Gamma：
|      ▲
|     /|\
|    / | \
|___/__|__\___ → 行权价
       ATM
       
0DTE 的 Gamma 更集中、更高！
```

### 这意味着什么？

1. **价格在 ATM 附近变化更敏感**
2. **做市商的对冲需求更大**
3. **GEX 效应在 0DTE 日更显著**

## 🎮 0DTE 交易方式

### 方式 1：买方（Long）

```
买 Call：看涨
买 Put：看跌

优点：损失有限（最多亏权利金）
缺点：Theta 快速吃掉你
适合：预期大波动，方向明确
```

### 方式 2：卖方（Short）

```
卖 Call：不看涨
卖 Put：不看跌

优点：收取时间价值，Theta 帮你赚钱
缺点：损失可能无限（需要保证金）
适合：预期小波动，区间震荡
```

### 方式 3：价差（Spread）

```
Call Spread / Put Spread：
同时买卖不同行权价的期权

优点：风险有限，成本低于单腿
缺点：利润也有限
适合：有方向观点但想控制风险
```

## ⚠️ 0DTE 风险管理

### 核心原则

```
1. 每笔交易风险 ≤ 账户的 1-2%
2. 不要全仓做 0DTE
3. 设好止损，严格执行
4. 不要报复性交易
```

### 资金管理示例

```
账户总资金：$10,000

每笔风险上限：$100-200 (1-2%)

如果买 $2 的 0DTE Put：
→ 最多买 50-100 张
→ 但建议只买 10-20 张，留安全边际
```

## 🕐 0DTE 的时间特性

### 一天中的 Theta 衰减

```
开盘时：Theta 衰减温和
        ↓
午餐时：衰减加速
        ↓
下午：  衰减更快
        ↓
尾盘：  ATM 期权价值快速归零
```

### 最佳交易时段

| 时段 (EST) | 特点 | 建议 |
|-----------|------|------|
| 9:30-10:00 | 开盘波动大 | 观望或小仓位 |
| 10:00-12:00 | 相对稳定 | 适合建仓 |
| 12:00-14:00 | 午餐时段清淡 | 少操作 |
| 14:00-15:30 | 下午趋势 | 适合交易 |
| 15:30-16:00 | 尾盘剧烈 | 谨慎或平仓 |

## 🔗 0DTE 与 GEX 的关系

0DTE 期权因为 Gamma 极高，对 GEX 影响巨大：

```
0DTE 占比高时：
→ Total GEX 更容易变化
→ 翻转（The Flip）更常发生
→ 关键价位更"有效"
```

这就是为什么我们用 GexBot 数据来指导 0DTE 交易！
